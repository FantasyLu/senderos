## 路线概述

```
土库曼斯坦：
  达瓦扎燃烧坑（"地狱之门"）
  → 阿什哈巴德（大理石之城，骏马大会）
  → 卡拉库姆沙漠（骑马穿越）
  → 马雷（古丝路重镇木鹿）
  → 土库曼巴希（里海港口）

哈萨克斯坦：
  阿克套（里海石油城）
  → 阿拉尔 + 咸海（环境灾难现场）
  → 阿克苏-热巴格雷国家公园（生态探访）
  → 图尔科巴斯（传统马球）
  → 卡拉干达（斯大林流放者之城）
  → 阿斯塔纳（未来主义新首都）
  → 塞米巴拉金斯克（苏联核试验区）
  → 阿拉木图（前首都）

塔吉克斯坦：
  杜尚别（165m 世界最高旗杆）
  → 安索布隧道（"死亡隧道"）
  → 苦盏（第二大城）
  → 库尔干秋别
  → 雅格诺布山谷（内战幸存者）
  → 霍罗格（直升机飞越帕米尔）
  → 布伦库尔村（海拔4000m，407人）
  → 中塔边境（瓦罕走廊）

吉尔吉斯斯坦：
  比什凯克（新娘绑架习俗采访）
  → 波孔巴耶沃（传统养鹰人）
  → 红色阵线村（德意志流放社区）
  → 阿尔斯兰博布（野生胡桃林）
  → 奥什（2010年族群冲突）
  → 贾拉拉巴德（结核病诊所）

乌兹别克斯坦：
  塔什干（中亚铁路终点站）
  → 努库斯（萨维茨基美术馆）
  → 阿姆河流域（河流衰竭）
  → 希瓦（丝路古城穿越时光）
  → 撒马尔罕（帖木儿帝国）
  → 布哈拉（蓝色穹顶）
  → 马尔吉兰（手工丝绸作坊）
```
