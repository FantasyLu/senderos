## 路线概述

```
太湖（1607）
  → 山东（泰山、曲阜、峄山）1609
  → 浙江（宁波、普陀山、天台山、雁荡山）1613
  → 安徽/福建（白岳、黄山、武夷山）1616
  → 皖赣（黄山再游、庐山）1618
  → 福建（九鲤湖）1620
  → 中原（嵩山、华山、武当山）1623
  → 福建、北京、盘山 1628–1629
  → 晋北（五台山、恒山）1633
  → 大西南四年壮游 1636–1640
    （浙→赣→湘→桂→黔→滇）
    终点：腾冲（最西端，98°E）
  → 昆明得病，轿送返江阴 1640–1641
  → 江阴病逝，时年 54 岁
```
