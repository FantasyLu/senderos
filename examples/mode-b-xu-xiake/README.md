## 路线概述

```
江阴故里（1607）
  → 苏南（惠山、太湖）1607
  → 山东（泰山、曲阜、峄山）1608–1609
  → 北京 1608
  → 浙江（宁海、天台山、雁荡山）1613
  → 皖闽（黄山、齐云山、武夷山）1616–1617
  → 浙江（普陀山）1617
  → 中原（嵩山）1621 / 盘山（天津）1621
  → 晋北（五台山、恒山）1622–1623
  → 西北（华山）1623 / 湖北（武当山）1623
  → 闽（九鲤湖、玉华洞）1629
  → 赣（庐山）1632
  → 大西南四年壮游 1636–1640
      湖南（衡山、永州）
    → 广西（兴安灵渠、桂林、阳朔、梧州、柳州、南宁）
    → 贵州（黎平、都匀、贵阳、安顺、黄果树）
    → 云南（曲靖、抚仙湖、石林、昆明、楚雄、大理、
            鸡足山、保山、腾冲、高黎贡山、丽江、
            玉龙雪山、泸沽湖、云龙、剑川、凤庆）
  → 昆明得病，轿送返江阴 1640–1641
  → 江阴病逝，时年 54 岁
```
