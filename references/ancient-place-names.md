# 古地名 → 现代坐标映射表

> 数据来源：历史地理研究文献、谭其骧《中国历史地图集》等权威资料。
> 格式：`古地名 | 现代地名 | 纬度 | 经度 | 朝代/时期 | 备注`
> 坐标精度：市级中心点（±0.5°），不代表精确遗址位置。

---

## 使用说明

`geocode.py` 在调用 OSM Nominatim 之前，先查本表。
命中规则：
1. `古地名` 完全匹配 → 直接使用坐标
2. 未命中 → 尝试去掉「府/州/路/郡/县」后缀再查
3. 仍未命中 → 调用 Nominatim API 查 `现代地名`
4. 全部失败 → 标记为 `unresolved`

### 古地名后缀对照
- `府` = 明清地方行政单位（相当于地级市）
- `州` = 比府小的行政单位（相当于县级市或县）
- `县` = 最基层行政单位
- `路` = 宋元行政单位（相当于省）
- `道` = 唐代行政单位（相当于省）
- `卫` = 明代军事行政单位

- 所有坐标为 WGS84 坐标系（与 OpenStreetMap 兼容）

- 部分古地名（如玉门关、阳关）为遗址考古坐标
